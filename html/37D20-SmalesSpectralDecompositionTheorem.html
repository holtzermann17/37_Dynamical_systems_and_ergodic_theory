Smale’s spectral decomposition theorem


Let M be a compact differentiable manifold and let f:M→M be an Axiom A diffeomorphism. The nonwandering set Ω of f can be partitioned into a finite number of compact topologically transitive blocks, called basic blocks:

Ω=⋃i=1mΛi.

Moreover, each basic block is partitioned into a finite number of compact subblocks Λi⁢j, j=1,…,mi such that f⁢(Λi⁢j)=Λi⁢(j+1) for 1≤j<mi and f⁢(Λi⁢mi)=Λi⁢1, and Λi⁢j is topologically mixing for fmi.
