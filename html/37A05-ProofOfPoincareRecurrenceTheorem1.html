proof of Poincaré recurrence theorem 1


Let An=∪k=n∞f-k⁢E. Clearly, E⊂A0 and Ai⊂Aj when j≤i. Also, Ai=fj-i⁢Aj, so that μ⁢(Ai)=μ⁢(Aj) for all i,j≥0, by the f-invariance of μ. Now for any n>0 we have E-An⊂A0-An, so that

μ⁢(E-An)≤μ⁢(A0-An)=μ⁢(A0)-μ⁢(An)=0.

Hence μ⁢(E-An)=0 for all n>0, so that μ(E-∩n=1∞An)=μ(∪n=1∞E-An)=0. But E-∩n=1∞An is precisely the set of those x∈E such that for some n and for all k>n we have fk⁢(x)∉E. □
