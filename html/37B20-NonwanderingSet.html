nonwandering set


Let X be a metric space, and f:X→X a continuous surjection. An element x of X is a wandering point if there is a neighborhood U of x and an integer N such that, for all n≥N, fn⁢(U)∩U=∅. If x is not wandering, we call it a nonwandering point. Equivalently, x is a nonwandering point if for every neighborhood U of x there is n≥1 such that fn⁢(U)∩U is nonempty. The set of all nonwandering points is called the nonwandering set of f, and is denoted by Ω⁢(f).

If X is compact, then Ω⁢(f) is compact, nonempty, and forward invariant; if, additionally, f is an homeomorphism, then Ω⁢(f) is invariant.
